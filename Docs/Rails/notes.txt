ERB stands for Embedded Ruby

----------------------------------------------------------------

rails new MyApp
stt MyApp/
cd MyApp
bundle install --without=production
rails g model Doctor first_name:string last_name:string specialty:string postal_code:integer
rails g model Appointment date:datetime doctor:references
rails db:migrate

heroku create thp-login
git add .
git commit -m "Project initialization"
git push heroku master
